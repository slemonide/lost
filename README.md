# lost

![Game screenshot](screenshot.png)

A maze exploring game.

## Gameplay
Avoid spikes and ghosts and collect coins.

Use TTS notifications with care.

## Controls

- w/a/s/d, arrow keys - movement
- +/- zoom in/out
- q/esc - exit the game
